## Logo Rules

✅ **DO:**
- Use full logo + wordmark for official communications
- Keep clear space of 8px minimum around logo
- Use on Sky White or Deep Ink backgrounds
- Maintain aspect ratio

❌ **DON'T:**
- Rotate, skew, or manipulate
- Change colors
- Remove the wordmark
- Place on conflicting backgrounds

## Hover States & Transitions

All interactive elements: **200ms ease-out**

### Standard Hover Pattern
1. Top border changes to 2px Gold
2. Slight upward translation (2px)
3. Light shadow appears

### Button Hover
- Primary: Darker Indigo + Gold border-top
- Secondary: 2px border + Gold top

## Common Do's & Don'ts

✅ **DO:**
- Use plenty of whitespace
- Stick to Indigo + Gold palette
- Keep designs minimal and clean
- Use uppercase for labels (letter-spacing: 1-2px)
- Test contrast for accessibility (4.5:1 minimum)

❌ **DON'T:**
- Use gradients (solid colors only)
- Add drop shadows everywhere
- Mix in corporate colors (blue, red)
- Use more than 3 colors per section
- Center all text (use asymmetry)
- Add complexity to icons

## Accessibility Checklist

- [ ] All text has 4.5:1 contrast ratio (minimum)
- [ ] Interactive elements have visible focus states
- [ ] Images have alt text
- [ ] Color is not the only way to convey information
- [ ] Hover states also visible on keyboard focus
- [ ] Line height minimum 1.5 for body text
